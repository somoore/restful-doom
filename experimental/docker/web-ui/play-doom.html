How to play using VNC:

1. When the VNC viewer loads, click inside the display area to focus it

2. Press F8 to access the noVNC menu and make sure "Enable keyboard" is checked

3. Use the arrow keys to move, Ctrl to fire, Alt to strafe, and Spacebar to open doors

4. If keyboard input isn't working, try clicking inside the game view again

How to use the Dashboard:

1. The dashboard shows a real-time view of DOOM plus API control buttons

2. Use the directional buttons to move and turn

3. Use the action buttons to shoot, use items/doors, etc.

4. If controls aren't working, check browser console for errors

5. For detailed API documentation, visit API Documentation

Troubleshooting:

VNC Issues: Make sure to click inside the game area to focus it.

API Control Issues: If dashboard controls aren't working, check the browser console for error messages.

Custom WAD Support: If you built the container with your own WAD file, it will be loaded automatically.
